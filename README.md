# 🎗️ Breast Cancer Detection — AI Web App

An AI-powered breast cancer detection web application built with Flask and Random Forest classification, trained on the Wisconsin Breast Cancer Dataset.

## 📊 Model Performance
| Metric | Score |
|--------|-------|
| Accuracy | 96.49% |
| Precision (Malignant) | 100% |
| Recall (Malignant) | 90% |
| F1 Score | 96% |
| AUC | ~0.99 |

## 🚀 Deploy on Render (via GitHub)

### Step 1 — Push to GitHub
```bash
git init
git add .
git commit -m "Initial commit: Breast Cancer Detection App"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/breast-cancer-detection.git
git push -u origin main
```

### Step 2 — Deploy on Render
1. Go to [render.com](https://render.com) and sign in
2. Click **"New"** → **"Web Service"**
3. Connect your GitHub repository
4. Render will **automatically detect** `render.yaml` and configure everything
5. Click **"Create Web Service"**
6. Wait ~3-5 minutes for the build to complete
7. Your app will be live at `https://breast-cancer-detection.onrender.com`

> ✅ The `render.yaml` file handles everything:  
> - Installs dependencies  
> - Runs `train.py` to generate `model.pkl`, `scaler.pkl`, and all visualizations  
> - Starts the Flask app with Gunicorn  

## 💻 Run Locally
```bash
# Install dependencies
pip install -r requirements.txt

# Train the model & generate visualizations
python train.py

# Start the app
python app.py
```
Open [http://localhost:5000](http://localhost:5000)
