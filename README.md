# 🎗️ Breast Cancer Detection — AI Web App

An AI-powered breast cancer detection web application built with Flask and Random Forest classification, trained on the Wisconsin Breast Cancer Dataset.

---

## 📊 Model Performance

| Metric | Score |
|--------|-------|
| Accuracy | 96.49% |
| Precision (Malignant) | 100% |
| Recall (Malignant) | 90% |
| F1 Score | 96% |
| AUC | ~0.99 |

---

## 📁 Project Structure

```
breast-cancer-detection/
├── app.py                        # Flask web server
├── train.py                      # Model training script
├── index.html                    # Frontend UI (responsive)
├── data.csv                      # Wisconsin Breast Cancer Dataset
├── model.pkl                     # Trained model (generated by train.py)
├── scaler.pkl                    # Feature scaler (generated by train.py)
├── requirements.txt              # Python dependencies
├── render.yaml                   # Render deployment config
├── .gitignore
└── static/
    └── visualizations/
        ├── confusion_matrix.png
        ├── roc_curve.png
        ├── feature_importance.png
        ├── correlation_heatmap.png
        └── class_distribution.png
```

---

## 💻 Run Locally

```bash
# Install dependencies
pip install -r requirements.txt

# Train the model & generate visualizations
python train.py

# Start the app
python app.py
```

Open [http://localhost:5000](http://localhost:5000)

---

## 🔬 Features

- 30 tumor measurement features (mean, SE, worst)
- Real-time prediction with confidence score
- Benign / Malignant classification
- 5 interactive model visualizations
- Fully responsive on mobile and desktop
- Sample data buttons for quick testing
